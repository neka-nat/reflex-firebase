# reflex-firebase

A Reflex custom component firebase.

## Installation

```bash
pip install reflex-firebase
```

## Getting Started

```bash
cp .env.sample .env

# FIREBASE_API_KEY=
# FIREBASE_AUTH_DOMAIN=
# FIREBASE_PROJECT_ID=
# FIREBASE_STORAGE_BUCKET=
# FIREBASE_MESSAGING_SENDER_ID=
# FIREBASE_APP_ID=
# FIREBASE_MEASUREMENT_ID=
```
